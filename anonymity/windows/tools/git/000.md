
## Github 访问慢的解决方案




到 https://www.ipaddress.com/ 找到这五个网址对应的IP地址
```
140.82.114.9 codeload.github.com
140.82.113.4 github.com
199.232.5.194 github.global.ssl.fastly.net
52.216.107.6  s3.amazonaws.com
52.217.36.188  github-cloud.s3.amazonaws.com
```


更改 `C:\Windows\System32\drivers\etc\hosts` 文件，在文件中追加上述五行，最后执行 `ipconfig /flushdns` 命令，刷新 DNS 缓存。
