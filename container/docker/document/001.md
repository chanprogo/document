
# 纵观 Docker - 运维视角




在运维视角中，主要包括  

1. 下载镜像、运行新的容器、  
2. 登录新容器、在容器内运行命令，  
3. 以及销毁容器。  








<br><br><br>

安装 Docker 的时候，会涉及两个主要组件：
Docker 客户端 和  
Docker daemon（ 也被称为“ 服务端” 或者“ 引擎”）。daemon 实现了 Docker 引擎的 API。

使用 Linux 默认安装时，客户端与 daemon 之间的通信    是通过本地 IPC/ UNIX Socket 完成的（`/var/run/docker.sock`）；  






可以使用 docker version 命令    来检测客户端和服务端    是否都已经成功运行，并且  可以互相通信。
